分类页面的筛选功能
	*传递的参数定义
		1.操作类型：
			-字段类型:  int 
			-字段名称:  operation
			-字段值：   全部=1   手柄=2  遥控=3  空鼠=4
		2.游戏语言
			 -字段类型: int 
			 -字段名称：language
			 -字段值：  全部=1   中文=2   英文=3
		3.游戏大小
			-字段类型	int
			-字段名称	size
			-字段值     全部=1  100M以下=2 300M以下=3 500M以下=4 1G以下=5 1G以上=6
		4.游戏类别
			-字段类型   int
			-字段名称   category
			-字段值     值就是之前分类中的类别ID
		5.游戏排序方式
			-字段类型   int
			-字段名称   rank
			-字段值     下载次数排序=1  更新日期排序=2
		6.GPU
			-字段类型   String 
			-字段名称   gpu
		7.页数
			-字段类型   int 
			-字段名称   pagenum
		8.页记录数
			-字段类型   int 
			-字段名称   pagesize
